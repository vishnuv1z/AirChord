# Chord Audio Files

Put one `.wav` file here for each chord in the wheel:

- `C.wav`
- `G.wav`
- `D.wav`
- `A.wav`
- `E.wav`
- `Am.wav`
- `Em.wav`
- `Dm.wav`

You can use short guitar chord samples for the best response. I have already included some sample audio files, but you can replace them or add any other chord audio files you want.
